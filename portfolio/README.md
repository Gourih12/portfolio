### Installation
1. Navigate to the portfolio directory:
```bash
cd portfolio
```
2. Install dependencies:
```bash
npm install
```


3. Start the development server:
```bash
npm run dev/ build


```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
### Add Images
- Replace placeholder images in `public/images/projects/`
- Add icons in `public/images/icons/`
- Add hero images in `public/images/hero/`

### Add Resume
- Place your resume PDF as `public/resume/Gouri_Resume.pdf`


## Building for Production
```bash
npm run build
npm start
```
